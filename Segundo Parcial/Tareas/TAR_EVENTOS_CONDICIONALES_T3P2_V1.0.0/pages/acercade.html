Aplicación de Eventos Relevantes en Elementos de Formulario con Condicionales if y if-else

Objetivo General:
Desarrollar una página web donde se utilicen todos los tipos comunes de elementos de formulario HTML junto con sus eventos más relevantes, y aplicar sentencias condicionales if y if-else en JavaScript para generar interacciones prácticas.

Instrucciones Específicas:
Incluir al menos un ejemplo por cada tipo de input o elemento de formulario, con su respectivo manejo de eventos.
Para cada evento relevante que se aplique al elemento, deberás:
Usar sentencias if y/o if-else.
Mostrar una respuesta visual en la página ( innerHTML, cambiar color, mostrar/ocultar, etc.).
Puedes usar funciones en JavaScript, pero todo el código debe estar en archivos externos, usar buenas practicas.
