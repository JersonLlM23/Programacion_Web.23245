Etiqueta blockquote

La etiqueta <blockquote> se utiliza para definir una sección de texto que es una cita de otra fuente. Generalmente, el texto dentro de un bloque de cita se muestra con sangría.

A continuación se muestra un ejemplo de uso de la etiqueta blockquote:

"La imaginación es más importante que el conocimiento. El conocimiento es limitado. La imaginación rodea el mundo."
