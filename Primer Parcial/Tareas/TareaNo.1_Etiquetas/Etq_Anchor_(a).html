Etiqueta Anchor

La etiqueta anchor se utiliza para crear enlaces a otras páginas web o a secciones dentro de la misma página.

Se representa con la letra "a" y su atributo más importante es el "href", que indica la URL a la que se enlaza.

Ejemplo de etiqueta anchor:

Visita facebook: Haz clic aquí para entrar a Facebook

Envíame un correo: Correo
