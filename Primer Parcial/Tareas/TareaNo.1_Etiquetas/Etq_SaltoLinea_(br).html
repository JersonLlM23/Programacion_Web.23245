Etiqueta br

La etiqueta se utiliza para insertar un salto de línea en el texto. Es útil cuando deseas dividir el contenido en líneas separadas sin crear un nuevo párrafo.

A continuación se muestra un ejemplo de uso de la etiqueta br:

Este es un ejemplo de texto con
un salto de línea.

Este es otro ejemplo de texto con
dos saltos de línea.
Y aquí hay otro salto de línea.

Recuerda que la etiqueta br es un elemento vacío, lo que significa que no tiene una etiqueta de cierre.
