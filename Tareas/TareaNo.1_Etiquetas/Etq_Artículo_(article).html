Etiqueta Article

La etiqueta article representa un contenido independiente y autocontenido que tiene sentido por sí solo.

Se utiliza para marcar secciones de un documento que pueden ser distribuidas o reutilizadas de forma independiente.

Por ejemplo, un artículo de un blog, una noticia o una entrada de un foro.

A continuación se muestra un ejemplo de uso de la etiqueta article:

Noticia 1

Se anunció una nueva versión de HTML.

Noticia 2

CSS ahora tiene nuevas funciones para diseños responsivos.

En este ejemplo, cada noticia es un artículo independiente que puede ser leído y entendido por sí mismo.
