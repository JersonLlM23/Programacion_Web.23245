Etiqueta Área del mapa de Imagen.

La etiqueta area se utiliza para definir áreas dentro de una imagen que pueden ser clicables.

Se utiliza en conjunto con la etiqueta map, que define el mapa de imagen.

La etiqueta area no es visible, pero permite crear enlaces en diferentes partes de una imagen.

Ejemplo:

Kai, ninja del Fuego
